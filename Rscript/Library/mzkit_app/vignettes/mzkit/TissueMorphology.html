TissueMorphology

require(mzkit);

#' spatial tissue region handler
imports "TissueMorphology" from "mzkit";

spatial tissue region handler tissue morphology data handler for the internal bionovogene MS-imaging analysis pipeline.

.NET clr type export
umap_point: UMAPPoint 3d scatter data point, a spatial spot or a single cell data


.NET clr function exports
tag_samples Add tissue region label to the pixels of the ms-imaging data
intersect_layer extract the missing tissue pixels based on the ion layer data
UMAPsample create a collection of the umap sample data
TissueData create a collection of the tissue region dataset
writeCDF export the tissue data as cdf file
loadTissue load tissue region polygon data
loadUMAP load UMAP data
read.spatialMapping read spatial mapping data of STdata mapping to SMdata
splitMapping Split the spatial mapping by tissue label data
gridding create a spatial grid for the spatial spot data
